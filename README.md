# Homework Management System

A comprehensive homework management system designed for teachers to assign and manage homework for students from Class 5 to 12 across multiple subjects.

## Features

### For Students
- 🎯 **Filter Homework** - By class, subject, question type, and week
- 📚 **Multiple Subjects** - Maths, Physics, Chemistry, Biology
- 📝 **Question Types** - MCQs, Short Answer (SAQ), Very Short Answer (VSAQ), Long Answer (LAQ)
- 🖼️ **Image Support** - Questions can include images
- 📅 **Weekly Organization** - Homework organized by weeks
- 🔍 **Quick Preview** - Click on any homework to view full details

### For Teachers/Admin
- ➕ **Add Homework** - Create new assignments with all details
- ✏️ **Edit Homework** - Update existing assignments
- 🗑️ **Delete Homework** - Remove assignments
- 📤 **Image Upload** - Attach images to questions
- 📊 **Manage All Classes** - Support for Class 5 to 12
- 💾 **JSON Storage** - Easy to backup and modify

## Technology Stack

- **Backend**: Python Flask
- **Frontend**: HTML5, CSS3, JavaScript
- **Storage**: JSON files (no database required)
- **Styling**: Modern CSS with Flexbox/Grid
- **Icons**: Font Awesome 6
